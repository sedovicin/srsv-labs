Program za simulaciju dva lifta odredjenih kapaciteta (jedan veci, drugi manji) s nekoliko katova. Putnici su nasumice generirani s odredjenom vjerojatnoscu.
Program je pisan u programskom jeziku Java, verzije 8.
U direktoriju src nalazi se izvorni kod programa. 
Main metoda nalazi se u klasi hr.fer.srsv.lab4.Main.java.
