Program za simulaciju jednostavnog križanja, odnosno dvije prometne trake (iz svakog smjera po jedna) sa jednim pješačkim prijelazom koji također ima dvije trake (za svaki smjer pješaka po jedan).
Program je pisan u programskom jeziku Java, verzije 8.
U direktoriju src nalazi se izvorni kod programa. 
Main metoda nalazi se u klasi hr.fer.srsv.lab1.Lab1Main.java.
